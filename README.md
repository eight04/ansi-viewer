ANSI Viewer
===========

A Firefox addon which let you view .ans file in the browser.

Features
--------

* Builtin Big5-UAO. Thanks for [BBSFox-E10S](https://github.com/ettoolong/BBSFox-E10S)!
* ANSI Color.
* Half CJK coloring.
* Draw header.
* Invert color with Alt+L. (Use CSS filter)
* Flashing text.
